# CareerCraft: AI-Powered Career Guidance Platform

## Problem Statement
CareerCraft is an AI-powered platform designed to provide comprehensive career guidance to users at various stages of their professional journey. It addresses the challenges of skill assessment, resume building, mentor matching, interview preparation, job searching, course recommendations, and college admissions prediction by leveraging machine learning and natural language processing.

## User Personas
- **Recent Graduate:** Needs help with resume building, skill assessment, and job searching.
- **Career Changer:** Requires course recommendations, mentor matching, and interview simulation.
- **Student:** Interested in college admission predictions and skill development roadmaps.
- **Professional:** Seeks advanced mentor matching and job board filtering.

## Wireframes Description
- **Homepage:** Navigation bar, hero section with platform overview, feature highlights.
- **Skill Assessment:** Form for user skills input, results page with scores and recommendations.
- **Resume Builder:** Template selection, NLP-powered content suggestions, preview and download.
- **Mentor Matching:** Profile input, recommendation list with filters.
- **Interview Simulator:** Chatbot interface for practice questions, voice integration.
- **Job Board:** Search filters, scraped job listings.
- **Course Recommender:** Goal-based form, ML-suggested courses.
- **Admission Predictor:** Academic profile input, prediction results.

## Tech Stack
- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **AI/ML:** scikit-learn, TensorFlow, NLTK, HuggingFace Transformers
- **Database:** SQLite (for simplicity, can upgrade to PostgreSQL)
- **Deployment:** Render or Heroku
- **Other Tools:** VS Code, GitHub, Postman for testing

## Features
1. **Skill Assessment (ML-based):** Analyzes user skills and provides career recommendations.
2. **Resume Builder (template + NLP):** Generates resumes with AI suggestions.
3. **Mentor Matching (recommendation engine):** Matches users with suitable mentors.
4. **Interview Simulator (chatbot + voice):** Simulates interviews with AI.
5. **Job Board (scraping + filtering):** Aggregates and filters job listings.
6. **Smart Course Recommender:** Suggests courses based on goals and skills.
7. **College Admission Predictor:** Predicts admission chances using ML.

## Installation
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python run.py`

## Usage
- Start the Flask server and access via browser.
- Navigate through features using the UI.
